# TraiNotes: Personal Workout Platform

This Android application, developed in **Java** using Android Studio, forms part of a full-stack training log system powered by an ASP.NET Core Web API backend. The platform enables authenticated users to track, record, and organize their workouts through a seamless mobile interface — add exercises grouped by category, set weights and repetitions, and keep everything synchronized with the server.

## Features

- Access your training notes through a user-friendly mobile app that communicates directly with the server
- Manage your personal profile — update your name, avatar, subscription preferences, password, and switch between accounts effortlessly
- View your training history with a built-in calendar — each day contains its own workout log and performed exercises
- Add and fully customize your exercises, organized into categories, with personal instructions and a favorites list for quick access
- Specify weight and repetitions for each new set, and organize them into clear, easy-to-read lists
- Save all changes to the server, ensuring your workouts remain accessible from any device at any time
- Customize your experience by adjusting the app language, weight units, date format, and other locale-based settings stored on your device
